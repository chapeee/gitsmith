# Contributing to gitsmith

Thanks for your interest in contributing.

## Development Setup

1. Install Node.js 20+
2. Clone the repo
3. Run:

```bash
npm install
npm link
```

## Running Locally

From this repo:

```bash
npm run lint
```

From a separate test repo:

```bash
gitsmith init
git add .
gitsmith
```

## Pull Request Guidelines

- Keep PRs focused and small.
- Add/update docs when behavior changes.
- Include clear reproduction steps for bug fixes.
- Keep user-facing errors friendly and actionable.

## Commit Style

Use this tool to create conventional commits according to the current project config.

## Reporting Issues

Use the issue templates for bug reports and feature requests.
